# Cyber Tanks
### Code Version 2.

1.0.2-a

## Building

Run `npm run build`.

## Deploying

Run `npm start`.

## Environment Variables

process.env.PORT - Sets the server port.

process.env.SALT - The salt for admin security.

process.env.ADMIN_KEY - SHA256 hex hash of admin password+salt.
